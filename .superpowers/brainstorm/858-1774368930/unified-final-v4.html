Implementation notes

  • All sizes in rem - no hardcoded px. Uses LUSENA's 1rem = 10px base (Dawn's 62.5%).
  • "In cart" tile width: flex: 0 0 30% - percentage-based, scales with container.
  • "Add" tile width: flex: 1 - takes remaining space (~70%).
  • Cart page layout identical to drawer - same card, same tiles, same proportions. The upsell zone sits within the cart items column width, not full page.
  • Cart page zone padding uses the page's horizontal padding (2.8rem) to stay aligned with cart items above.
