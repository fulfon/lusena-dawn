Button hierarchy

Primary (filled) - leave cart

Secondary (outline) - bundle

Secondary (outline) - cross-sell

Rule: Filled teal = leave the cart. Outline teal = stay and add something. Text differentiates the action.

Changes in v2

  • Bundle CTA → outline teal (was filled). Now subordinate to checkout.
  • "Add" tile meta text removed. Just the product name - clean, no competing text sizes.
  • "In cart" tile keeps "W koszyku" status. Functional asymmetry: settled item vs action item.
  • Consistent button component. Same outline style, different text: "Dodaj" vs "Dodaj do zestawu".
