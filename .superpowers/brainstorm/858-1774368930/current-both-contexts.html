What we see

  • Cross-sell looks clean in both contexts - balanced 3-column card
  • Bundle nudge is crammed into the aside column - dashed box with full-width button doesn't fit the card layout
  • Drawer constraint - only ~335px usable width, the bundle nudge overflows the aside area
  • No savings visibility on cross-sell - bundle nudge shows "zam. 508 zl" but regular cross-sell has no value framing
  • Different mental models - "Dodaj" (add to cart) vs "Dodaj do zestawu" (swap/upgrade) need different visual treatment but within the same design system
