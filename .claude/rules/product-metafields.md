# Product Metafields

Before creating or modifying product metafields, read `memory-bank/doc/products/product-metafields-reference.md` - it contains universal fields (specs, feature cards 2/4/6, care) shared across ALL products that must NOT be modified per product. Only fields marked "REQUIRES CREATIVE SESSION" in the product file should be crafted (including feature card 5, which is product-specific). Never use percentage claims for momme comparisons (e.g., "30% gestszy") - use qualitative language only.
